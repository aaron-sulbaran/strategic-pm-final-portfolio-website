# North Star Metric

## Apple Wallet

**Author:** Aaron Sulbaran

## North Star Metric

**Weekly Active Credential Uses per User (WACU/U):** the average number of successful credential interactions (payments, ID presentations, transit taps, key unlocks, boarding passes, health insurance presentations) completed per active Wallet user per week.

**Category:** Transaction. Wallet wins when each tap delivers value at the point of need, not when users spend time inside the app. Time-in-app is an anti-signal for this product; if a user is looking at Wallet, the experience has already failed.

## Supporting Input Metrics

WACU/U is moved by three input clusters that map directly to the levers the Wallet team can pull through the roadmap:

| Input Cluster | Example Metrics | What It Drives |
|---|---|---|
| **Breadth** | Full-suite credential adoption rate (3+ types), Health credential activation rate, Onboarding completion rate | More credentials in Wallet means more daily moments of use |
| **Reliability** | Credential surfacing accuracy, NFC tap success rate, International transaction success rate | Fewer failed taps means users trust Wallet over the physical card |
| **Frequency** | Average uses per credential per week, % of users with 5+ uses/week, Physical wallet abandonment rate | Repeated, habituated use across daily life occasions |

## Why This Drives Business Results

- **Customer-centric, not vanity.** Counts moments where users got real value. Downloads, sign-ups, and credentials added are acquisition metrics, not value metrics, and the article explicitly warns they create the illusion of growth while customer value flatlines.
- **Habit-forming by design.** Repeat use is exactly what the metric measures, so growth in WACU/U means Wallet is genuinely replacing the leather wallet in daily life. This is the sustained-value test the Sean Ellis framework requires.
- **Leading indicator of long-term Apple business outcomes.** Higher WACU/U produces stickier iPhone owners (lower churn to Android), more Apple Pay interchange revenue, and a deeper switching-cost moat as users add more credentials they cannot easily move off the platform.
- **Aligned with the existing PRD strategy.** The success metrics already defined in the Apple Wallet PRD (full-suite adoption, surfacing accuracy, transaction success, health credential activation) all sit naturally as input metrics under WACU/U, giving the team a single number to rally around without rewriting the strategy.
- **Actionable at the team level.** Engineering can move reliability inputs, design can move frequency inputs through contextual surfacing, and partnerships can move breadth inputs by signing new issuers, transit agencies, and insurers. Every roadmap item has a clear path to the North Star.
